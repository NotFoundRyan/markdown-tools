# Images Directory

This directory contains screenshots for the README.

## Required Images

Please add the following screenshots:

1. `screenshot-home.png` - Main homepage screenshot (recommended size: 800x600 or larger)

## How to Add Screenshots

1. Take a screenshot of the application
2. Save it with the appropriate filename in this directory
3. The README will automatically display the image

## Notes

- Use PNG format for best quality
- Keep file sizes reasonable (< 1MB recommended)
- Use descriptive filenames
