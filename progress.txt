# Markdown to Word 项目开发日志

## 初始化记录
- 时间: 2026-04-04
- 项目名称: markdown-to-word
- 项目目标: 创建本地 Web 应用，支持 Markdown 与 Word/HTML/PDF 之间的相互转换
- 技术栈: Vite + TypeScript + 纯 CSS

## 需求概述
1. 功能页面:
   - Markdown 转 DOCX
   - Word 转 Markdown
   - Markdown 转 PDF
   - Markdown 转 HTML
   - HTML 转 Markdown
   - 更新日志页面

2. 技术要求:
   - 文件夹分类规范，CSS、HTML 分离
   - i18n 国际化支持（中英文）
   - 明暗风格支持，跟随系统
   - 导航栏
   - 黑白灰风格
   - 磨砂玻璃质感
   - 弹窗背景模糊
